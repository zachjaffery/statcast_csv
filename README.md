# stat-cast_csv

### Requirements
pandas is used. Run:
> pip3 install pandas
> 
### Usage
In terminal, navigate to folder containing the code

For example:
> cd ~/downloads/statcast_csv-main

Then, run `python3 getstats.py {TEAM1} {TEAM2} {PLAYER_TYPE}`
A second team is not required, and by default "player type" is batter.

Ex:
> python3 getstats.py LAD NYY
